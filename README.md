# SaaS Inventory & Offer Management System

A full-stack SaaS-style application for managing products, inventory and offers with multi-tenant architecture.

The project focuses on clean database design, business logic and scalable architecture.

## Tech Stack

- TypeScript
- Next.js
- React
- Node.js
- PostgreSQL
- Prisma ORM
- Tailwind CSS
- Docker

## Features

- Multi-tenant architecture (Company based isolation)
- RBAC (Role Based Access Control)
- Product management
- Offer creation with multiple items
- Snapshot pricing for historical consistency
- Relational database design with PostgreSQL
- REST API with Next.js
- Email verification via Brevo SMTP

## Architecture

The system follows a SaaS multi-tenant architecture.
